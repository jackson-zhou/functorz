Functorz Studio 是一款可视化低代码页面编辑器，支持拖拽组件布局、属性编辑、事件流程绑定和小程序预览构建。整体采用 React + TypeScript 构建，使用 Zustand + Immer 管理状态，@dnd-kit 实现拖拽，@antv/x6 实现流程图编辑。

5 核心组件详解 — App

5.1 状态与副作用

useEffect触发条件行为
auto-saveproject / saveState 变化调用 scheduleAutoSave，200ms 防抖保存到 IndexedDB

flowEventTypeselected 变化Page 默认 'load'，其他默认 'tap'

5.2 DnD 拖拽体系

使用 @dnd-kit/core 提供拖拽能力，支持两种拖拽模式：

  • 新建 从左侧组件面板拖入画布 → insert 命令
  • 移动 在画布/组件树中拖拽节点 → move 命令
sequenceDiagram
    participant P as 组件面板 PaletteItem
    participant D as DndContext
    participant C as 画布 WebNode
    participant S as Store

    P->>D: onDragStart(componentType)
    D->>D: 设置 dragPreview 状态
    Note over D: 显示拖拽预览图标

    P->>C: onDragOver (进入画布)
    C->>C: 显示 drop-target 高亮

    C->>D: onDragEnd (在画布上松开)
    D->>S: execute({ type:'insert', componentType, parentId, index })
    S->>S: 更新 project 树
    S-->>D: 返回新状态
    D->>D: 显示 toast 通知

    Note over P,C: 节点间拖拽（move 模式）同理
    P->>D: onDragStart(nodeId)
    D->>S: execute({ type:'move', nodeId, parentId, index })

renamePage(id, name)重命名页面双击编辑提交
